# Customer Support Analytics Dashboard

## 📊 Project Overview
This project analyzes customer support ticket data to identify trends in resolution time, customer satisfaction, and operational efficiency.

## 🛠 Tools Used
- Power BI
- SQL
- Python (Google Colab)

## 📁 Dataset
Synthetic dataset with 1000+ records including:
- Priority
- Channel
- Region
- Agent
- Resolution Time
- Customer Satisfaction

## 🔍 Key Insights
- High-priority tickets show higher escalation rates
- Chat channel resolves tickets faster
- Certain regions generate more tickets
- SLA breaches observed in longer resolution cases

## 📈 Dashboard Features
- KPI Cards
- Ticket Distribution Analysis
- Resolution Time Analysis
- Customer Satisfaction Analysis
- Agent Performance Tracking
- Interactive Slicers

## 📎 Files Included
- `.pbix` → Power BI dashboard
- `.csv` → Dataset
- `.sql` → Queries used
- `.pdf` → Dashboard export

## 🚀 Outcome
This project demonstrates data analysis, visualization, and business insight generation using Power BI.
